# Excel 毕业率计算器 - 计算逻辑分析与基础数据提取指南

## 一、毕业率定义

**毕业率 = ADPS / 目标ADPS**

其中目标ADPS是一个固定值（如鸣金影为 `143152.08`），代表该职业流派在105阶竞速环境下的理论毕业DPS。

---

## 二、核心计算链路

```
毕业率 = I16 = I12 / 目标ADPS
    ↓
ADPS = I12 = I10 / I8
    ↓
总伤 = I10 = SUM(L:L)  (所有技能行的伤害求和)
    ↓
战斗时间 = I8 = 固定值（如94秒）
    ↓
单技能伤害 = L = M*BO + N*BM + O*BP + P*BQ
    ↓
M/N/O/P = 基于属性的伤害公式
    ↓
BO/BM/BP/BQ = 命中概率分布
```

## 四、伤害计算核心公式拆解

### 4.1 单技能伤害 = L = M*BO + N*BM + O*BP + P*BQ

其中：
- **BO** = 精准命中概率
- **BM** = 会心命中概率
- **BP** = 会意命中概率
- **BQ** = 普通命中概率

### 4.2 命中概率计算

```
BJ = 精准率 = MIN(IF(强制精准=1, 100%, 面板精准率), 100%)

BK = 会心率 = IF(强制会心=1, 100%, MIN(80%, 面板会心率+技能会心率+增益会心率) + 直接会心率+增益直接会心率) * IF(强制会意=1, 0, 1)

BM = 会意率 = IF(强制会意=1, 100%, MIN(40%, 面板会意率+技能会意率+增益会意率) + 直接会意率+增益直接会意率) * IF(强制会心=1, 0, 1)

BO = MIN((1-BM)*BJ, BJ*BK)
BP = 1 - BM - BO - BQ
BQ = IF(强制会心=1, 0, (1-BJ)) * (1-BM)
```

### 4.3 伤害值计算 (M/N/O/P)

以M为例（精准/普通命中的基础伤害）：

```
M = [
    ((AO+AP)/2 * AI + AJ) * (1+AQ) * (1+AR)  // 外功伤害部分
    + ((AT+AU)/2 * IF(武学属性="鸣金", AK, AI) + IF(武学属性="鸣金", AL*1.5, 0)) * (1+AV) * (1+AW)  // 鸣金属性伤害
    + ((AX+AY)/2 * IF(武学属性="裂石", AK, AI) + IF(武学属性="裂石", AL*1.5, 0)) * (1+AZ) * (1+BA)  // 裂石属性伤害
    + ((BB+BC)/2 * IF(武学属性="牵丝", AK, AI) + IF(武学属性="牵丝", AL*1.5, 0)) * (1+BD) * (1+BE)  // 牵丝属性伤害
    + ((BF+BG)/2 * IF(武学属性="破竹", AK, AI) + IF(武学属性="破竹", AL*1.5, 0)) * (1+BH) * (1+BI)  // 破竹属性伤害
] * (1+AM) * BL * Q * (1+IF(定音="流血", 流血定音, 0)) * (1+BR)
```

其中各中间变量：

| 变量 | 公式 | 说明 |
|------|------|------|
| AM | = G3 + 套装增伤 + 技能通用增伤 + SUM(增益通用增伤) | 总通用增伤 |
| AO | = D10*(1+AN) + 技能最小外功 + SUM(增益最小外功) - 怪物防御 | 实际最小外功 |
| AP | = E10*(1+AN) + 技能最大外功 + SUM(增益最大外功) - 怪物防御 | 实际最大外功 |
| AN | = 技能外功加成 + SUM(增益外功加成) | 外功加成 |
| AQ | = IF(外功穿透-外抗+技能穿透+增益穿透>0, (值)/200, (值)/100) | 外功穿透收益 |
| AR | = 外攻伤害加成 + 技能外攻加成 + SUM(增益外攻加成) | 外攻伤害加成 |
| AS | = 技能属性加成 + SUM(增益属性加成) | 属性攻击加成 |
| AT/AU | = 属性最小/最大攻击*(1+AS) | 实际属性攻击 |
| AV | = IF(属性穿透-属抗+技能穿透+增益穿透>0, (值)/200, (值)/100) | 属性穿透收益 |
| AW | = 属性伤害加成 + 技能属性加成 + SUM(增益属性加成) | 属性伤害加成 |
| AI | = 技能外功倍率*(1+IF(定音="流血", 固伤加成/10, 0)) | 外功倍率 |
| AJ | = 技能外攻固伤*(1+IF(定音="流血", 固伤加成/10, 0)) | 外攻固伤 |
| AK | = 技能属性倍率 | 属性倍率 |
| AL | = 技能属性固伤*(1+IF(定音="流血", 固伤加成/10, 0)) | 属性固伤 |
| BL | = 1 + 会心伤害加成 + 技能会心伤害 + SUM(增益会心伤害) | 会心伤害倍率 |
| BN | = 1 + 会意伤害加成 + 技能会意伤害 + SUM(增益会意伤害) | 会意伤害倍率 |
| BR | = SUM(增益特殊增伤) | 特殊增伤 |
| Q | 技能释放次数 | |

### 4.4 N（会心伤害）与M的区别

N使用会心伤害倍率BN代替BL，且使用最大攻击值(AP)代替平均值。

### 4.5 O（会意伤害）与M的区别

O使用会意伤害倍率BN代替BL。

### 4.6 P（强制命中伤害）

P是简化版，不含命中概率相关的BL/BN倍率。

## 六、计算流程伪代码

```python
def calculate_graduation_rate(player, config, monster, rotation, skills_db, buffs_db, target_adps):
    total_damage = 0

    for action in rotation:
        skill_name = action["skill"]
        count = action["count"]
        active_buffs = action["buffs"]
        skill = skills_db[skill_name]

        # 1. 计算实际属性面板
        actual_stats = calculate_actual_stats(player, config, monster, skill, active_buffs, buffs_db)

        # 2. 计算命中概率
        hit_probs = calculate_hit_probabilities(player, skill, active_buffs, buffs_db)

        # 3. 计算各命中类型的伤害
        damage_precise = calculate_damage(actual_stats, skill, "precise")
        damage_critical = calculate_damage(actual_stats, skill, "critical")
        damage_conscious = calculate_damage(actual_stats, skill, "conscious")
        damage_normal = calculate_damage(actual_stats, skill, "normal")

        # 4. 单技能总伤害
        skill_damage = (
            damage_precise * hit_probs["precise"] +
            damage_critical * hit_probs["critical"] +
            damage_conscious * hit_probs["conscious"] +
            damage_normal * hit_probs["normal"]
        ) * count * skill["真气比例"]

        total_damage += skill_damage

    # 5. 计算ADPS和毕业率
    adps = total_damage / config["战斗时间"]
    graduation_rate = adps / target_adps

    return graduation_rate
```

## 七、各流派差异

不同流派（鸣金影、破竹尘、裂石威、裂石钧、牵丝霖、破竹鸢、鸣金虹）的主要差异：

1. **目标ADPS不同**：每个流派有各自的毕业标准值
2. **武学属性不同**：主属性不同（鸣金/裂石/牵丝/破竹）
3. **技能轴不同**：各流派有不同的技能释放序列
4. **增益组合不同**：触发的增益效果组合不同
5. **武学奇术表不同**：各流派技能的伤害倍率不同

但**核心计算逻辑完全相同**，只是输入数据不同。
