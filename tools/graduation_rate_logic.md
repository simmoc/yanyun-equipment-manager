# 燕云十六声 各流派毕业率计算逻辑

> 本文档基于 base_excel 目录下的各流派竞速轴属性毕业率计算器（包含 100 级版本和 105 阶进阶版本）整理而来。

---

## 一、数据分类

### 1.1 固定数据（不随装备变化）

| 类别 | 内容 |
|------|------|
| 怪物防御参数 | 外抗（20/36）、外防（559）、属抗（24/40），普通/精英模式不同 |
| 技能倍率表 | 每个技能的外功倍率、外攻固伤、属性倍率、属性固伤、蓄力标记 |
| 增益效果表 | 每种 buff 对会心率/会意率/通用增伤/穿透等属性的加成值 |
| 会心/会意基础加成 | 会心伤害基准、会意伤害基准（105阶通用）|
| 技能轴 | 各流派固定的技能序列与每种技能的使用次数 |
| 毕业档 DPS 基准值 | 用于计算毕业率的分母（各流派不同）|

### 1.2 用户输入数据（随装备变化）

| 数据 | 说明 |
|------|------|
| 各武器最小/最大攻击 | 外功、鸣金、裂石、牵丝、破竹、无相 |
| 各武器穿透值 | 影响穿透增伤系数 |
| 各武器伤害加成 | 特定武器类型的加成 |
| 精准率 | 触发精准判定的概率 |
| 会心率、会意率 | 会心/会意触发概率 |
| 会心伤害加成、会意伤害加成 | 会心/会意时的额外伤害比 |
| 通用增伤 | 全局伤害乘区加成 |
| 蓄力增伤 | 蓄力技能的额外加成 |
| 首领增伤 | 对首领的额外增伤 |
| 武器增伤（伞/扇等）| 特定武器触发的增伤 |
| 战斗时间（秒）| 用于计算 DPS |
| 激活的增益列表 | 当前战斗中生效的 buff 组合 |

### 1.3 输出数据

| 输出 | 说明 |
|------|------|
| 各技能期望伤害 | 单次技能的期望值（考虑会心/会意概率） |
| 总期望伤害 | 整个技能轴的总期望 |
| DPS（每秒伤害）| 总期望 / 战斗时间 |
| **毕业率** | 当前 DPS / 毕业档 DPS × 100% |
| 各武器伤害占比 | 外功/属性攻击的贡献分布 |

## 二、核心计算公式

### 2.1 穿透增伤系数

穿透增伤反映装备穿透属性对怪物防御的突破效果：

```
穿透增伤 = 1 + (穿透值 - 怪物防御值) / 100
```

- 穿透值 > 怪物防御：穿透增伤 > 1（正增益）
- 穿透值 < 怪物防御：穿透增伤 < 1（减伤）

怪物防御参数（105阶）：

| 模式 | 外防（外抗） | 属防（属抗） |
|------|-------------|-------------|
| 普通 | 559（20） | 559（24） |
| 精英 | 559（36） | 559（40） |

> 穿透增伤 = 1 + (穿透 - 抗性值) / 100

### 2.2 单技能伤害计算（105阶）

每个技能按攻击类型（外功/属性）分开计算，汇总得到期望伤害：

```
单次伤害期望 = 会心期望 × 会心率 + 会意期望 × 会意率 + 普通期望 × 普通占比 + 擦伤期望 × 擦伤占比
```

各命中类型计算：

| 命中类型 | 计算公式 |
|----------|---------|
| **会心** | `max攻 × 倍率 + 固伤 × (1 + 会心伤害加成) × (1 + 外攻加成) × (1 + 外攻增伤) × 穿透增伤 × 总增伤系数` |
| **会意** | `max攻 × 倍率 + 固伤 × (1 + 会意伤害加成) × (1 + 外攻加成) × (1 + 外攻增伤) × 穿透增伤 × 总增伤系数` |
| **普通** | `avg攻 × 倍率 + 固伤 × (1 + 外攻加成) × (1 + 外攻增伤) × 穿透增伤 × 总增伤系数` |
| **擦伤** | `min攻 × 倍率 + 固伤 × (1 + 外攻加成) × (1 + 外攻增伤) × 穿透增伤 × 总增伤系数 × 0.5` （注：部分流派无擦伤） |

其中：
- `avg攻 = (min攻 + max攻) / 2`
- `总增伤系数 = (1 + 通用增伤) × (1 + 蓄力增伤 × 是否蓄力技) × (1 + 首领增伤)`

### 2.3 外功与属性攻击混合计算

由于技能可以同时受外功和属性攻击加成，期望伤害实际上是：

```
技能期望 = 外功部分 + 属性部分
外功部分 = (最小外功 ~ 最大外功) × 外功倍率 + 外攻固伤
属性部分 = 属性攻击(主武器类型) × 属性倍率 + 属性固伤
```

如果本流派的主武器是牵丝，则属性倍率会乘以牵丝攻击：

```
属性攻击 = 当前激活流派的武器攻击值
```

### 2.4 会心/会意概率分配

精准率决定命中条件，会心/会意在命中框架内计算：

```
// 精准率处理（105阶简化）
实际精准率 = 输入精准率  // 通常已含装备精准

// 概率分配
擦伤占比 = 1 - 精准率
会心占比 = 精准率 × 会心率
会意占比 = 精准率 × 会意率
普通占比 = 精准率 × (1 - 会心率 - 会意率)
```

### 2.5 增益叠加

每个技能有固定的增益列表，各增益的属性加成**叠加到对应字段**：

```
通用增伤（累加）= 基础通用增伤 + Σ(各增益.通用增伤)
外攻穿透（累加）= 基础穿透 + Σ(各增益.外功穿透)
会心伤害（累加）= 基础会心伤害加成 + Σ(各增益.会心伤害)
属性穿透（对应流派）= 基础属性穿透 + Σ(各增益.对应流派穿透)
```

增益中有"通用增伤"字段的直接加入通用增伤乘区：
- `伞增` → 通用增伤 +0.24
- `四时之秋` → 通用增伤 +0.10
- `沧海帖10层` → 会心伤害 +0.10
- `易水歌5层` → 通用增伤 +0.05, 外功穿透 +10

### 2.6 总期望与 DPS

```
总期望 = Σ (技能期望[i] × 次数[i])
DPS = 总期望 / 战斗时间（秒）
```

### 2.7 毕业率

```
毕业率 = DPS / 毕业档DPS × 100%
```

毕业档 DPS 是 Excel 中设定的参考基准值，代表"满装备"状态下的 DPS：

| 流派 | 毕业档 DPS（普通目标） |
|------|---------------------|
| 牵丝霖_105 | ~89783 |
| 其他105阶流派 | ~80085 |
| 牵丝玉_100 | ~114168 |

## 三、各流派差异说明

### 3.1 牵丝霖（105阶）

- 主武器：牵丝，使用枪、剑等多属性武器
- 套装：撼天（提供通用增伤加成约 33.3%）
- 技能轴：神龙吐火 + 太白醉月组合
- 特色：蓄力技太白醉月5享受蓄力增伤加成

### 3.2 裂石（威/钧）（105阶）

- 主武器：裂石
- 套装：通用（通用增伤约 26.8%）
- 技能轴：以裂石属性技能为主

### 3.3 鸣金（影/虹）（105阶）

- 主武器：鸣金
- 套装：通用（通用增伤约 26.8%）
- 技能轴：以鸣金属性技能为主
- 特色：部分技能有 1.5 倍固伤加成

### 3.4 破竹（尘/鸢）（105阶）

- 主武器：破竹
- 套装：通用（通用增伤约 26.8%）
- 技能轴：以破竹属性技能为主

### 3.5 100级版本差异

- 战斗时间更长（76秒 vs 15秒）
- 参数区较简单（武器奇术sheet独立，技能轴更少）
- 装备参数用目标属性（笃山衔蝉等）评估

## 四、计算流程图

```
用户输入装备属性
    ↓
加载流派配置（技能轴、增益列表、怪物参数）
    ↓
合并增益效果（通用增伤、穿透、会心伤害等）
    ↓
对每个技能计算：
  ├─ 外功部分期望（外功攻击 × 外功倍率 + 外攻固伤）
  ├─ 属性部分期望（属性攻击 × 属性倍率 + 属性固伤）
  ├─ 穿透增伤系数
  ├─ 增伤乘区（通用增伤、首领增伤、蓄力增伤）
  └─ 会心/会意/普通/擦伤期望加权
    ↓
Σ(技能期望 × 次数) = 总期望伤害
    ↓
DPS = 总期望 / 战斗时间
    ↓
毕业率 = DPS / 毕业档DPS × 100%
```
